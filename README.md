# work-tracker
מחשבון שכר ומעקב שעות עבודה בעיצוב מודרני ורספונסיבי. האפליקציה מאפשרת הזנה מהירה של שעות עבודה (כולל תמיכה במשמרות לילה), בחירת תעריף שעתי ושמירת הנתונים בזיכרון המקומי. היא כוללת לשונית היסטוריה מפורטת עם סיכומי שכר שבועיים וכלליים, אפשרות למחיקת רשומות וממשק משתמש (UI) נקי המותאם לשימוש נוח מהנייד.
